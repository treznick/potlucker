/rest/users

  • GET: lists all users in JSON
  • GET {key}/{value}: gets a user by key and value
  • POST /create : creates a user, must have:
    • a value named "name." This must be a string.
    • a pantry array named "pantry". This must be stored as a ; delineated string
  • GET /delete/{key} : deletes a user by its primary key

/rest/recipes

This is based on the Yummly API. For more information please visit https://developer.yummly.com/

  • GET /{ingredients} : gets recipes by an ingredient string, which is a whitelist string separated by ;
